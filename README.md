# PackWalk

PackWalk is a realtime dog-walking meetup platform. Users can go live on the map while walking, control who can see them, join public or private groups, and coordinate meetups through chat and live ETA awareness.

## Workspace Layout

- `apps/web` - Next.js web application and future PWA shell
- `apps/mobile` - reserved for future Expo / React Native app
- `packages/shared` - shared types, validation, and utility code
- `docs/product` - product scope, user flows, and feature notes
- `docs/architecture` - technical decisions and system design notes
- `supabase` - schema, migrations, and backend setup
