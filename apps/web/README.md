# PackWalk Web

Next.js web app for the PackWalk MVP.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- TanStack Query
- Zustand
- Supabase
- Mapbox

## Development

From the repo root:

```bash
npm run dev:web
```

Or from this folder:

```bash
npm run dev
```

## Environment

Copy `.env.example` to `.env.local` and fill in:

- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- `NEXT_PUBLIC_MAPBOX_TOKEN`
