# PackWalk User Flows

## 1. Onboarding

The user creates an account, completes a profile, adds at least one dog, and chooses default visibility preferences.

## 2. Start A Walk

The user taps `Go for a walk`, selects:

- who can see the walk
- exact or approximate location sharing
- how long the walk should remain active

The user then appears on the live map according to those rules.

## 3. Discover Walkers

Another user opens the map, sees nearby active walkers they are allowed to view, opens a profile preview, and can follow, message later, or invite them into a group context.

## 4. Create Or Join A Group

The user creates a public or private group, invites people, or joins an existing public group to coordinate a meetup.

## 5. Coordinate In Real Time

Group members exchange messages, share live status, and see who is closer or how long others may take to arrive.

## 6. End A Walk

The walk ends either manually or automatically when it expires. The user disappears from live discovery, while chat and group history remain available.
