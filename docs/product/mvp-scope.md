# PackWalk MVP Scope

## Product Direction

PackWalk is a live meetup-first platform for people walking with dogs. The core experience is discovering who is walking now, coordinating meetups, and controlling who can see your live walk.

## In Scope For v1

- Account registration and sign-in
- User profile
- Dog profile basics
- Start walk action
- Live walk session with visibility settings
- Map view with active walkers
- Follow system
- Public and private groups
- Group chat
- Distance and ETA awareness for active group members
- Notifications for walk starts, invites, and chat events

## Out Of Scope For v1

- Payments
- Ads or monetization
- AI route planning
- Pet services marketplace
- Advanced moderation tooling
- Smartwatch integrations
- Matching by breed, temperament, or behavior

## Core Product Rules

- A user appears on the live map only during an active walk session.
- Visibility is user-controlled per walk session.
- The app must support approximate location sharing.
- Groups can be public or private.
- Real-time chat and live presence are core features, not optional extras.

## Primary Screens

- Authentication
- Home
- Live Map
- Group Details
- Group Chat
- Profile
- Notifications
- Privacy Settings
